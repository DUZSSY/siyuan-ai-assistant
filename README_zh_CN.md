# AI助手插件

**专注于文本块级 AI 编辑的思源笔记插件** —— 选中即编辑，所见即所得。

## ✨ 核心功能

- 📝 **文本块级 AI 编辑**：选中任意文本块，一键润色、翻译、总结
- 💬 **直接对话功能**：通过自然语言与AI对话，自定义修改文本
- 🌊 **流式响应**：实时逐字显示AI生成内容，支持推理模型思维链展示
- 📜 **操作历史**：完整记录文本处理过程，支持版本时间线和diff对比，支持↩️撤销恢复上一次操作快照
- 🤖 **多AI提供商支持**：OpenAI、Ollama、DeepSeek、Claude等
- ✨ **8种智能文本处理**：润色、翻译、总结、扩写、精简、改写、续写、撤销
- 🔧 **5个自定义按钮**：配置专属AI操作按钮和提示词
- 📊 **可视化 Diff 对比**：清晰展示修改内容（删除标红、新增标绿）
- 🎯 **精准文本替换**：支持全量块快照级回退保护
- 🎨 **智能浮动工具栏**：选中文本自动弹出，支持一键撤销变更
- 🔒 **隐私优先**：API密钥本地存储，支持Ollama本地部署

## 🚀 快速开始

### 方式1：浮动工具栏（推荐）

1. 在思源笔记中选中任意文本
2. 浮动工具栏自动出现在选中文本附近
3. 点击需要的 AI 操作：
   - **快捷操作**：润色、翻译、总结、扩写、精简、改写、续写
   - **💬 对话**：打开对话界面，用自然语言指令修改文本
   - **自定义按钮**：你配置的专属AI操作
4. 在 Diff 弹窗中查看修改，确认后应用

**Diff 查看器功能**：
- 逐条接受/拒绝修改，或一键应用全部
- 直接编辑修改后的内容再应用
- 根据自定义要求重新生成

**对话模式**：
- 输入自然语言指令（如"精简一下"、"增加技术细节"）
- 查看历史对话，恢复之前的会话
- 界面内置快捷操作按钮

### 方式2：右键菜单

1. 右键点击块图标
2. 选择 AI 操作
3. 查看 Diff 对比，确认后应用

### 方式3：对话面板

1. 点击浮动工具栏上的"💬 对话"按钮
2. 输入自然语言修改要求
3. 在 Diff 查看器中查看结果并应用

## ⚙️ 配置指南

### 推荐：Ollama 本地部署（完全免费，隐私最佳）

1. 安装 [Ollama](https://ollama.com)
2. 拉取模型：
   ```bash
   ollama pull qwen3.5:4b
   # 或
   ollama pull gemma4:e4b
   ```
3. 插件设置中选择 Ollama 提供商
4. 使用默认配置即可（API地址：http://localhost:11434）

**🌐 云模型推荐**（2026年7月最新）：
- **日常使用（润色/翻译/对话）**：GPT-5.3 Instant, Claude Sonnet 5, GLM-4.7-Flash, Qwen3.5:4B, gemma4:e4b
- **专业写作（复杂推理）**：GPT-5.5 Thinking, Claude Opus 4.7, GLM-5.2, Qwen3.6
- **极致性能（代码/复杂任务）**：GPT-5.5 Pro, Claude Sonnet 5, Kimi K3, DeepSeek V4-Pro
- **本地部署（隐私优先）**：Ollama → Qwen3.6 27b（首选）, DeepSeek V4-Flash（高配）, LLaMA4 系列

> 💡 以上模型可通过兼容 OpenAI API 的服务使用。Qwen3.5:4b 是本地部署的平衡之选。

### 其他AI提供商

支持以下AI服务：
- **OpenAI** (GPT-5.5 Pro ⭐推荐 / GPT-5.5 Thinking / GPT-5.3 Instant)
- **Claude** (Claude Sonnet 5 ⭐推荐 / Claude Opus 4.7)

在插件设置中添加对应API密钥即可使用。

### 工具栏配置

自定义浮动工具栏中显示的按钮：

1. 打开插件设置 → **工具栏** 标签页
2. 启用/禁用标准操作按钮（润色、翻译等）
3. 在"自定义提示词"标签页配置的按钮会自动同步到这里

### 自定义按钮配置

创建最多5个个性化的AI操作按钮：

1. 打开插件设置 → **自定义提示词** 标签页
2. 启用并配置每个自定义按钮：
   - **按钮名称**：显示名称（如"转表格"）
   - **图标**：Emoji图标（如📊）
   - **AI提示词**：给AI的指令（如"将以下内容转换成表格形式"）
3. 保存后会自动同步到浮动工具栏

**💡 关于 Claude 支持**
Claude 通过 OpenAI API 兼容接口提供支持，配置方法：
- API 地址: `https://api.anthropic.com/v1`
- 模型: `claude-3-sonnet-20240229` 或 `claude-3-opus-20240229`
- 需申请 Anthropic API Key

## ⚠️ 数据安全风险提示

**🚨 重要提示：在使用AI操作功能前，请充分了解以下风险：**

### AI操作与文本块写入风险

- **AI可能产生意外结果**：AI可能会删除、重新排序或大幅改动您的原文内容。插件**无法保证**完全保留您的原始格式和内容。

- **块级替换风险**：应用AI生成内容时，整个文本块会被替换。如果AI输出不符合预期（如内容丢失、格式变化、意外删除），原始内容**无法自动恢复**。

- **多段内容处理**：AI可能生成多段内容，目前会写入单个文本块。应用前请仔细核对。

- **网络/API异常**：处理过程中可能因请求超时或API错误导致写入不完整或错误内容。

### 🛡️ 防护措施

1. **备份文档** — 对重要内容使用AI功能前，请先备份您的文档
2. **仔细核对** — 在Diff窗口中仔细核对AI输出，确认无误后再点击"应用修改"
3. **先测试** — 先在非关键内容上试用，了解AI对您写作风格的处理方式
4. **不确定时点"取消"** — 随时可以重新生成或放弃当前结果

> 💡 **建议**：对于重要文档，请在使用AI功能前从思源导出备份（文件 → 导出）。
